# Project Wiki

Welcome to this project's wiki — the home for human-readable documentation. This is a stub; the `/setup` phase populates it with a short project summary and links to the initial seed pages in each section.

> [!NOTE]
> This page is maintained by the `documenter` sub-agent. Humans may edit it directly; the sub-agent preserves human edits.

## How this wiki is organized

The sidebar groups every page into one of four sections:

### 🛠 Development
How to build, run, test, and contribute locally. Start here if you're a new contributor.

### 📟 Operational
How to run this project in production — deploy, observe, debug, roll back.

### 🎨 Design
Product and UX intent. Features, user flows, and the rationale behind non-obvious choices.

### 🏗 Architecture
How the system is built — subsystems, data flow, and decisions worth knowing.
